## Install

Installing the FiveM2Discord Resource is pretty simple just follow the steps below

---

<Steps>
    ### Download the resource
    Make sure you  grab the latest version from [here](https://github.com/toxic-development/FiveM2Discord/releases/latest)

    ### Extract the resource
    Unzip the download and extract it to your preferred location.

    ### Add the resource
    Drag/Drop the resource into your server resources folder.

    <Callout type="warning" emoji="⚠️">
       Make sure you place the `fivem2discord` folder in your server resources!
       ![Hmm](/examples/fivem2discord/project.png)
    </Callout>

    ### Ensure the resource
    Head over to your `server.cfg`, locate where you start/ensure your resources and add `ensure fivem2discord`
    ![Hmm](/examples/fivem2discord/ensure.png)

    ## Save and profit
    Finally save your changes to the server config and enjoy your new script!

</Steps>
